# Repositório do website das Pyladies São Paulo

Desenvolvimento do site da comunidade Pyladies na cidade de São Paulo.

# Estrutura do site

```bash
/
|--- Public # Pasta com conteúdo a ser rederizado

|     | 
|     |---  Style
|     |     | main.css
|     |

| 404.md
| index.html
| license
| readme.md 
```
